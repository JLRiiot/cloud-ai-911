## About the course

The course is divided into several modules, each covering a different aspect of cloud computing. Each module includes slides and exercises.

- [Module 1: Introduction to Cloud Computing](module-1-introduction-to-cloud-computing)
- [Module 2: Cloud Architecture](module-2-cloud-architecture)
- [Module 3: Cloud Migration](module-3-cloud-migration)
- [Module 4: Networking and Security](module-4-networking-and-security)
- [Module 5: Working with Servers and Microservices](module-5-working-with-servers-and-microservices)
- [Module 6: Real-Time Data Processing](module-6-real-time-data-processing)
- [Module 7: Introduction to AI and Machine Learning in the cloud](module-7-introduction-to-ai-and-machine-learning)
- [Module 8: Real-Time Image Processing](module-8-real-time-image-processing)
- [Module 9: IoT and Edge Computing](module-9-iot-and-edge-computing)
